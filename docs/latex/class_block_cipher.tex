\doxysubsection{Detailed Description}
Abstract base class for block ciphers. 

Block ciphers always operate in electronic codebook (E\+CB) mode. Higher-\/level classes such as C\+F\+B128 and C\+T\+R128 wrap the block cipher to create more useful classes for encryption and decryption of bulk data.
