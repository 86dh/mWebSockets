\hline

If {\itshape len} is less than \mbox{\hyperlink{class_s_h_a1_a69082dd1db04743cc4e0b7d9a34776d2}{hash\+Size()}}, then the hash value will be truncated to the first {\itshape len} bytes. If {\itshape len} is greater than \mbox{\hyperlink{class_s_h_a1_a69082dd1db04743cc4e0b7d9a34776d2}{hash\+Size()}}, then the remaining bytes will left unchanged.

If \mbox{\hyperlink{class_s_h_a1_a5a6a8a6169aa48e0bccadb22a149ab7c}{finalize()}} is called again, then the returned {\itshape hash} value is undefined. Call \mbox{\hyperlink{class_s_h_a1_ab71aaf39ed956320054861a2fbfa454f}{reset()}} first to start a new hashing process.
