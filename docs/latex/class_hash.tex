If {\itshape len} is less than \mbox{\hyperlink{class_hash_adcdd30de3e5ecaa2f798c0c5644d9ef8}{hash\+Size()}}, then the hash value will be truncated to the first {\itshape len} bytes. If {\itshape len} is greater than \mbox{\hyperlink{class_hash_adcdd30de3e5ecaa2f798c0c5644d9ef8}{hash\+Size()}}, then the remaining bytes will left unchanged.

If \mbox{\hyperlink{class_hash_a09b3ccec22763fc86b1415695862977c}{finalize()}} is called again, then the returned {\itshape hash} value is undefined. Call \mbox{\hyperlink{class_hash_a7b94309acaa5f52386785fb780e5be61}{reset()}} first to start a new hashing process.
